🛍️ 웹 페이지 연습 프로젝트

본 프로젝트는 HTML, CSS, JavaScript(jQuery)를 연습하기 위해 개인 학습용으로 제작한 웹 페이지입니다.
쇼핑몰 형태의 레이아웃을 구성하며 웹 페이지의 기본 구조와 스타일링, 간단한 동적 기능 구현을 연습하는 것을 목표로 했습니다.
실제 서비스 목적이 아닌 프론트엔드 기초 학습을 위한 연습용 프로젝트입니다.

📌 구현 내용

HTML을 이용한 페이지 구조 설계

CSS를 활용한 레이아웃 및 스타일링

jQuery를 이용한 동적 기능 구현

상단 메뉴 및 서브 메뉴

이미지 슬라이드 쇼

탭 메뉴 전환

쇼핑몰 메인 페이지 UI 구성 연습

🧠 학습 목적

웹 페이지 기본 구조(header, nav, section, footer) 이해

CSS 파일 분리 및 공통 스타일 관리 연습

JavaScript 파일 분리 및 기능별 스크립트 구성

jQuery를 활용한 DOM 제어 및 이벤트 처리 연습

✍️ 비고

본 프로젝트는 웹 개발을 한 연습용으로 제작되었습니다.
